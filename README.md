### 介绍
通过对接物联网设备，实时动态获取未来24小时附近海域潮汐数据
### 设计思路
通过购买数据接口，爬取所有接口的点位信息（含经纬度），在用户打开小程序的时候，获取用户当前位置经纬度，自动匹配最近的监测站点的潮汐数据信息进行展示（用户可手动切换位置）。
